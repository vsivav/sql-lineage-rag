# SQL Lineage RAG

Enterprise-grade SQL Server Lineage Engine with Retrieval-Augmented Generation (RAG).

## Features

- SQL Server Stored Procedure Parser
- Table Lineage
- Column Lineage
- CTE Lineage
- Temporary Table Lineage
- MERGE / UPDATE / INSERT / DELETE Analysis
- Expression Detection
- Transformation Lineage
- Metadata Generator
- FAISS Vector Store
- Local LLM Integration
- FastAPI
- Streamlit UI

## Technology

- Python 3.11+
- SQLGlot
- FAISS
- Sentence Transformers
- FastAPI
- Streamlit
